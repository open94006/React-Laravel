# React-Laravel CRUD 前後段分離

使用React.js來去呼叫api，Laravel的部分**只需要整理好資料**就可以了，在前端的部分React.js也能專心作呈現畫面的工作，**前後端分離的優勢**就出來了。 

- 一個**前後端分離**的簡單CRUD網站  
- 符合業界上的**職務分工**，理解前後端各自會遇到的問題，以及**需要溝通**的部分  
- 使用PHP Laravel結合前端框架React.js
